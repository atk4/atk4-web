Packages

  • atk4-sample-project-4.0.1.zip

    Download, unpack into the webroot and open in your browser. Sample project comes with both atk4 and atk4-addons and includes heavily documented files which will help you understand the structure of the project and get started. Recommended to new users.

  • atk4-sample-website-4.0.1.zip

    A really simple website using atk4. Download this if you are a designer or are just starting with PHP. This package starts with the basic stuff.

  • atk4-4.0.1.zip

    A stand-alone installation of Agile Toolkit core. Extract into your project directory. You can always replace your atk4 directory with more recent version.

  • atk4-addons-4.0.1.zip

    A stand-alone package containing some of the contributed addons for Agile Toolkit.
