Agile Toolkit features

Agile Toolkit is a PHP5 based framework developed by Agile Technologies. Toolkit includes a solid MVC framework, large number of ready-to-use widgets and controllers and are specifically designed for Web Software development in a rapid and Agile way.

Toolkit is split in 2 components:

  • Agile Toolkit Core - providing efficient implementation to any other frameworks
  • ATK Interface Library - introduces set of visual components and incredibly speeds up development of UI.

Features Benefits
1. Highly re-usable modules Typical Software as a Service (SaaS) application has 80% of code that overlays other similar applications.

Re-usage of components makes it:
  1. Quicker to make thus cheaper
  2. Less mistakes & more functionality
  3. Lets focus on the unique parts of your software
2. Front end interface design About 30% time and budget in each project is spend developing front end HTML & User Interface (UI). User Interface is most powerful selling point for SaaS applications so a lot of investment goes just to make project competitive.

Default UI benefits:
  1. Allows interface to be consistent over all application
  2. Developer can create simple application without involving designer or knowing HTML
  3. Follows design guidelines and standards
  4. Designer can develop templates that will be later applied to software
3. Security included Framework is built according to general practices and recommendations for secure software published by OWASP and takes responsibility of maintaining security away from developers.

What it means for developer:
  1. Don't have to worry about security vulnerabilities
  2. Less testing and audits involved
  3. Security standards are already in place
4. Business goal oriented design Agile Toolkit was built with a business owners in mind. Framework requires less training and has extensive documentations for every business feature you would need.

  1. Less training involved
  2. Business goals can be achieved in minutes
  3. Simple and extensive documentation
5. Newest technology Microsoft, Apple and Google are collaboratively promoting newest technological standards for web. For other companies to take advantage of those new technologies takes a lot of learning and getting used to.

Agile Framework allows you to:
  1. Produce great results with HTML5, JavaScript and CSS
  2. Includes technologies such as jQuery, jQuery UI and MVC
  3. Creates richer client-side software using AJAX technologies
6. Split between team members Typical web project would include architect, user interface developer, designer and business analyst. What if dependencies on each-other would be less critical?

Role separating between members mean:
  1. Less management and paperwork
  2. You don't have to put them in one room to deliver result
  3. Each person can focus on what they know best
7. Fast and highly scalable Modularity and Object-Oriented-Design of the software and frameworks enables dynamic activation of different components. For instance if user looks at his profile page, then code responsible for building "login" page is not used.

Dynamically loads visual elements such as Forms, Grids, Tables and Menus only if they are required. If AJAX is used to re-draw Form then other components are excluded. Similar approach works on frontend with JavaScript, jQuery and jQuery UI components loaded on demand. This makes software fast and efficient.
8. Integrated Many frameworks decide to have as less dependencies as possible. Using them often limits developer in terms of licensing and leaves a huge code footprint. In contract Agile Toolkit relies on standard components such as XML, DOM and XSLT to perform all the tasks. Most of the work is done by relevant PHP extensions.

  1. Make your solution highly customisable and extensive
  2. Easier to maintain in long term
  3. No licensing implications of 3rd party modules
9. Supported If you have ever used open source software, you should know that it always comes "as is". Creators of the software bear no warranty. Eventually software achieves good results (such as Linux, Apache or PHP itself), but it requires high level of control, management and at least 2 major rewrites.

Agile Toolkit has commercial license that allows us to hire developer team for polishing and perfecting our product. We can work on additional components which are in demand for commercial users and offer continuos growth and integration.
10. Licensing Agile Toolkit is licensed per developer per year. With introduction of cloud computing and the physical limits on CPU speeds, more and more people are looking for horizontal scaling - that is running more computers to do the job. We don't charge you per server and we don't charge you if you use our software to do open source projects. This is much reasonable licensing on longer run.
