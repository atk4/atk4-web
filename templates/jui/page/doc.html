Agile Toolkit is a high-performance web software framework written in PHP5. It introduces object-oriented web user interface designed for simplicity, customization and reusability. Agile Toolkit is ideal for commercial and open-source web software projects requiring interactivity, dynamics and unification of user interface.

Getting Started

Do you have few minutes to create your first Web Application using Agile Toolkit?

Armed with your own copy of freshly downloaded Agile Toolkit you are just a few minutes away making your first web software, even if you have never made web application before. You will need a text editor and LAMP compatible webserver.

The Community

Our belief is that the next generation of Web software wil be developed through commercial and open-source communities like ours. We are here to give you the core foundation of Agile Toolkit and make it available for both open-source and commercial projects. We also have provided some of the addons. What remains for you to do is to think of a great idea and have it working within weeks. Many have already did so, so come and share experience, suggestions and tips with us.

Agile Toolkit is popular with SMEs, Startups and other companies. We want all of you to be successful and invite different companies to collaborate and integrate their on-line software. By using same platform it's much easier. If you are building or planning to build a website using Agile Toolkit - let us know and we will be able to offer you a lot of competitive benefits.
