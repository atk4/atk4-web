Core Classes

  • Abstract Object — common ancestors for all classes
  • Base API classes — select the right API class for your application
    • Logger — Enhance your API with error and event display / logging
    • PathFinder — Resource locations, lookup and file paths
    • DBlite — Database Connectivity, regular and Dynamic Queries
    • SMlite — Lightweight template engine
    • Location — URL awareness, creating relative links and URL with arguments
    • Auth — User authentication
    • jQuery — jQuery and UI integration framework
    • Exception — Exception classes and handling
  • View — Standard Views
    • Form — UI Form class
    • Lister — User—defined lists
    • Grid — Multi—column type—aware tables
    • Info — Informative controls: warning, info and error boxes, icons
  • HtmlElement — Java—Script enabled controls
    • Button — Standalone Buttons, button enhancement
    • Layout — Laying out views on the pages: Tabs, Accordions and more
  • Non-visual supplementary frameworks
    • TMail — Template—based emails
    • ProcessIO — Interaction with system—processes in a secure way
    • HTMLSanitizer — Cleans up HTML you supply using XSLT
    • RecordOrder — Flexible way to change array order
    • VerionControl — SQL Database version control scripts and classes
  • MVC — Model View Controller
    • Model — Basic models and SQL Models
    • Controller — Interaction between Views and Models — Controllers
