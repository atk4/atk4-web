Installing Agile Toolkit

Agile Toolkit is designed to work as soon as you download it. You don't need to setup a database, any custom routing, or modules. Agile Toolkit does not use code generation. It is simple and straight-forward.

There are a lot of tweaking capabilities, but as with anything in Agile Toolkit, it's designed to 'just work' out-of-the-box.

Requirements

Agile Toolkit 4.2 requires PHP at least 5.3+ running on Windows or Linux. Third party add-ons may have additional requirements.
